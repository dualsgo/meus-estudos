Bem-vindo ao **Meus Estudos**. Este repositório foi criado para organizar seu aprendizado de forma lógica, moderna e escalável, unindo diversas plataformas e trilhas de conhecimento em um único lugar.

## 📂 Estrutura do Repositório

Aqui estão as principais trilhas de conhecimento contidas neste repositório:

- 🐍 **`Python`** - Notas de aulas exportadas, apresentações estruturadas e material essencial de Python.
- 📓 **`Resumos`** - Cadernos interativos em Jupyter Notebook (`.ipynb`) com resumos aprofundados dos conceitos.
- 🌐 **`cisco_fundamentos_python`** - Exercícios e conteúdos do curso Python Essentials (1 e 2) da Cisco.
- 🎓 **`cursoemvideo_python`** - Desafios e anotações organizados por "Mundos" do Curso em Vídeo.
- 📱 **`mimo_app`** - Aprendizado de HTML e Python através das rotas e desafios do aplicativo Mimo.
- ⚡ **`py-estudos`** - Um ecossistema React/Vite de um Dashboard moderno para leitura dinâmica e testes.

---

## 🖥️ Como Acessar o Dashboard Interativo

Para facilitar a navegação por todo esse conteúdo, desenvolvemos um **Dashboard Interativo Premium**.
Com ele, você pode buscar, explorar pastas e abrir arquivos em uma interface *Glassmorphism* super moderna.

**Para abrir o Dashboard:**
1. Navegue até a raiz deste repositório.
2. Clique ou abra o arquivo `index.html` no seu navegador de preferência.

> 🛠 **Se você adicionar novos arquivos ou cursos**, você pode atualizar o Dashboard executando o script de automação (`python build_dashboard.py`), que detectará todas as novas pastas e atualizará o portal!

## 🚀 Como Atualizar o Dashboard

Caso adicione novas pastas, você não precisa fazer o código do zero. Simplesmente rode:

```powershell
python build_dashboard.py
```
Esse script Python irá escanear o repositório inteiro e reconstruir a sua interface `index.html` instantaneamente com os novos dados.
