Isso garante que qualquer valor que você encontrar será maior ou menor que o valor inicial. # Em vez de digitar um valor arbritariamente grande ou pequeno, podemos usar os valores float('-inf') e float('inf') para representar o menor e o maior valor possível, respectivamente. Pense no maior valor que vier na sua cabeça. Agora, pense em um valor maior que esse. Não importa o valor que você pensou, o valor float('inf') é maior que ele.
