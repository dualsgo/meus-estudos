# Um loop é uma estrutura de controle que permite executar um bloco de código várias vezes. Em Python, existem dois tipos de loops: o for e o while.

# 2. O loop for itera sobre essa sequência de números, atribuindo cada número à variável i.
# 3. Dentro do bloco de código do loop, a função print("Olá, mundo!") exibe a mensagem "Olá, mundo!" na tela.

# O laço for é capaz de percorrer qualquer sequência de elementos, ou seja, qualquer objeto iterável. Isso inclui listas, tuplas, dicionários, conjuntos, strings, etc. A sintaxe do laço for é a seguinte:

# for elemento in sequência:
#     # bloco de código

# for é a palavra-chave que indica o início do laço.
# elemento é a variável de controle que armazena o elemento atual da sequência. A cada iteração, essa variável é atualizada com o próximo elemento da sequência, ou seja, ela assume o valor de cada elemento da sequência temporariamente (durante a iteração). Você pode dar o nome que quiser a essa variável, mas é recomendável escolher um nome que faça sentido em relação à sequência que está sendo percorrida.
# in é a palavra-chave que indica que a variável elemento irá percorrer a sequência. Ele é o operador de associação (você pode estudar mais sobre ele, pois é utilizado em diversas situações).
# sequência é a sequência de elementos que será percorrida. Pode ser uma lista, uma tupla, um dicionário, um conjunto, uma string, etc.

# O bloco de código é o conjunto de instruções que será executado a cada iteração do laço. Ele é executado para cada elemento da sequência.

# Vamos usar um exemplo muito simples para ilustrar o funcionamento do laço for. Suponha que temos uma lista de números e queremos exibir cada número na tela. Podemos fazer isso com um laço for:

numeros = [1, 2, 3, 4, 5] # lista de números

for numero in numeros: # para cada número na lista de números
    print(numero) # imprime o número
    
# Vou adicionar o mesmo exemplo com while para comparação. Note que diferente do for, precisamos iniciar um contador antes do loop e incrementá-lo a cada iteração. O for faz isso automaticamente.
# Além disso precisamos verificar a condição de parada. O for faz isso automaticamente, pois ele percorre a sequência até o final.
#  Para exibir o valor da lista, precisamos acessar o índice da lista. O for faz isso automaticamente, pois ele percorre a lista e armazena o valor atual em uma variável temporária.

numeros = [1, 2, 3, 4, 5] # lista de números
i = 0 # contador

while i < len(numeros): # enquanto contador for menor que o comprimento da lista de números
    print(numeros[i]) # imprime o número
    i += 1 # aumenta o contador
    
# Certo, agora suponha que alem do valor, eu precise saber a posição do valor na lista. Para isso, podemos usar a função enumerate().
# A função enumerate() é usada para percorrer uma sequência ao mesmo tempo em que mantém um contador. Ou seja, além do elemento atual, ela também retorna o índice desse elemento na sequência. Ela retorna uma tupla contendo o índice e o valor de cada elemento da sequência. A sintaxe da função enumerate() é a seguinte:

# for indice, elemento in enumerate(sequência):
#     # bloco de código

# indice é a variável que armazena o índice do elemento atual na sequência. Ou seja, 0 para o primeiro elemento, 1 para o segundo elemento, 2 para o terceiro elemento, e assim por diante.
# elemento é a variável que armazena o elemento atual da sequência. A cada iteração, essa variável é atualizada com o próximo elemento da sequência, da mesma forma que no laço for convencional, sem a função enumerate().

# enumerate(sequencia) é a função que retorna uma tupla contendo o índice e o valor de cada elemento da sequência e sequência é a sequência de elementos que será percorrida.

# Observação: A função enumerate() aceita um argumento adicional chamado start que define o valor inicial do contador. Por padrão, o contador começa em 0, mas você pode alterar esse valor se quiser.

# Vamos a um exemplo para ilustrar o uso da função enumerate(). Suponha que temos uma lista de nomes e queremos exibir cada nome juntamente com sua posição na lista começando em 1. 

nomes = ['Alice', 'Bob', 'Carol', 'David', 'Eve'] # lista de nomes

for indice, nome in enumerate(nomes, 1): # para cada índice e nome na lista de nomes, começando em 1
    print(f'{indice} - {nome}') # imprime o índice e o nome
