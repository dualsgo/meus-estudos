Para isso, usamos o fatiamento de strings. O fatiamento de strings é uma técnica que permite acessar partes de uma string. O fatiamento de strings é feito com colchetes e dois pontos. O primeiro número é o índice inicial e o segundo número é o índice final. Se o índice final for omitido, o fatiamento será feito até o final da string. Se o índice inicial for omitido, o fatiamento será feito a partir do início da string. Se ambos forem omitidos, o fatiamento será feito em toda a string. O terceiro número é o passo, que indica quantos caracteres serão pulados.
